# Security Policy

Please do not open public issues with cookies, Bearer tokens, Google account details, project IDs, generated private media, or local database contents.

If you find a security issue, report it privately to the repository maintainer.

Before sharing logs or screenshots, remove:

- Google account email addresses
- `Authorization` headers and Bearer tokens
- cookies and browser profile paths
- Flow project IDs and media IDs for private work
- local machine usernames and absolute paths
